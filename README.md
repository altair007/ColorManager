ColorManager
============

颜色管理,颜色名和对应Rgb值的快速查询
颜色管理 用于颜色英文名对应颜色RGB值的快速查询和修改.

可以从一个给定的.plist文件初始化√ 可以将颜色数据写入一个.plist文件中√

可以从任意文件读取数据,进行初始化,只要提供合理的block!√ 可以颜色名值对以任意方式写入任意类型的文件,只要提供合理的block!√

可以从一个给定的存颜色数据的字典初始化√ 可以以字典的形式返回所有的颜色数据.√

可以根据RGB数据,查询颜色的英文名称√ 可以根据英文名称,查询颜色的RGB数据√ 可以添加,删除,更改颜色的英文名称和RGB数据.√

使用委托机制,数据存储不依赖特定的NSDictionry类,任何符合ColorDictionary的类对象都可以用作ColorManager类的代理.√
